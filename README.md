# Director-Class AI

High-assurance multi-detector governance for autonomous / mission-critical
systems. A separate commercial product from the Apache-licensed `director-ai`
base: a parallel detector ensemble + action-plane effector kill-switch governing
three planes at once — content (is it true?), integrity (was the input tampered
with?), and action (is this command safe to execute?).

Design & plan: `docs/internal/` in the development tree.
